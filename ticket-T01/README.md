# Ticket T01 — Implementación UI (demo)

Descripción
- Página estática de demostración para el ticket T01. Incluye un diseño claro y moderno con efectos 3D sutiles, letras negras, marco azul y botones que enlazan páginas relacionadas.

Archivos
- `index.html` — Página principal del mockup.
- `styles.css` — Estilos (tema claro, 3D, azul).
- `related-1.html` y `related-2.html` — Páginas relacionadas a las que apuntan los botones.
- `images/product-1.svg` — Imagen de producto placeholder.

Cambios recientes

Cómo ver
- Abrir `ticket-T01/index.html` en un navegador moderno. En móvil se adapta a una columna.
